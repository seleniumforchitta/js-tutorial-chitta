# JavaScript Tutorial by Chitta

> A complete, project-driven JavaScript learning roadmap — from absolute beginner to expert level. Built as a beautiful dark-mode single-page website with structured curriculum, curated videos, and hands-on projects.

🌐 **Live Site:** [https://seleniumforchitta.github.io/js-tutorial-chitta/](https://seleniumforchitta.github.io/js-tutorial-chitta/)

---

## 📖 About

This is a free, open JavaScript learning resource designed to take anyone from writing their first variable all the way to understanding V8 engine internals and writing Babel plugins. Every phase is structured, sequenced, and paired with the best free videos and resources available on the internet.

No fluff. No paywalls. Just a clear path.

---

## ✨ Features

- 🌑 **Beautiful dark-mode UI** — custom-designed with a code-editor aesthetic
- 📋 **5 structured learning phases** — each with topics, videos, and projects
- 🎬 **Curated video links** — hand-picked from Fireship, Traversy Media, freeCodeCamp, and JSConf
- 💻 **Live code examples** — syntax-highlighted samples showing progression from Phase 1 to Phase 5
- 📦 **12+ hands-on projects** — real builds at every level, not just theory
- 📚 **Free resources section** — books, sites, YouTube channels, and official docs
- ⚡ **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step

## 🗺️ The Learning Roadmap

| Phase | Title | Level | Duration |
|-------|-------|-------|----------|
| 1 | Foundations of JavaScript | Beginner | 3–4 weeks |
| 2 | Core JS & the DOM | Elementary | 4–6 weeks |
| 3 | Async, OOP & APIs | Intermediate | 5–7 weeks |
| 4 | Patterns, Performance & Tooling | Advanced | 6–8 weeks |
| 5 | Expert-Level Mastery | Expert | Ongoing |

## 📚 Phase Breakdown

### Phase 1 — Foundations (Beginner)
**Topics:** Variables, data types, operators, control flow (if/else, switch), loops (for, while, for...of), functions, arrays, objects, typeof, type coercion

**Videos:**
- [JavaScript in 1 Hour — Programming with Mosh](https://www.youtube.com/watch?v=W6NZfCO5SIk)
- [Learn JavaScript — Full Beginner Course (freeCodeCamp)](https://www.youtube.com/watch?v=PkZNo7MFNFg)
- [The Modern JavaScript Tutorial — javascript.info](https://javascript.info)

**Projects:** Temperature converter · Simple quiz app · Console to-do list

---

### Phase 2 — Core JS & the DOM (Elementary)
**Topics:** Scope & hoisting, arrow functions, array methods (map/filter/reduce), destructuring, spread & rest, template literals, DOM manipulation, addEventListener, event delegation, error handling

**Videos:**
- [JavaScript Crash Course — Traversy Media](https://www.youtube.com/watch?v=hdI2bqOjy3c)
- [JavaScript DOM Manipulation — freeCodeCamp](https://www.youtube.com/watch?v=y17RuWkWdn8)
- [Array Methods You Must Know — Fireship](https://www.youtube.com/watch?v=aXOChLn5ZdQ)

**Projects:** Interactive to-do app · Expense tracker · Color palette generator

---

### Phase 3 — Async, OOP & APIs (Intermediate)
**Topics:** Callbacks, Promises (.then/.catch), Promise.all/race, async/await, Fetch API, REST APIs & JSON, OOP & classes, inheritance, closures, ES modules (import/export), localStorage

**Videos:**
- [Async JavaScript — Traversy Media](https://www.youtube.com/watch?v=PoRJizFvM7s)
- [Object Oriented JavaScript — freeCodeCamp](https://www.youtube.com/watch?v=Mus_vwhTCq0)
- [JavaScript Closures Explained — Fireship](https://www.youtube.com/watch?v=DHvZLI7Db8E)

**Projects:** Weather app (OpenWeatherMap API) · Movie search (OMDB API) · OOP bank account

---

### Phase 4 — Patterns, Performance & Tooling (Advanced)
**Topics:** Event loop & call stack, microtasks & macrotasks, design patterns, functional programming, currying & composition, generators & iterators, WeakMap/WeakSet, Web Workers, Service Workers, testing with Jest, Vite/Webpack, TypeScript basics

**Videos:**
- [What the heck is the event loop? — Philip Roberts (JSConf)](https://www.youtube.com/watch?v=8aGhZQkoFbQ)
- [JavaScript Design Patterns — Fireship](https://www.youtube.com/watch?v=jnME98ckDbQ)
- [TypeScript Tutorial — Programming with Mosh](https://www.youtube.com/watch?v=2pL28CcEijU)

**Projects:** Real-time chat app (WebSockets) · Jest-tested utility library · Infinite scroll with IndexedDB cache

---

### Phase 5 — Expert-Level Mastery (Expert)
**Topics:** V8 engine internals, JIT compilation, Proxy & Reflect, property descriptors, memory management & garbage collection, advanced async patterns, AbortController, WebAssembly interop, XSS/CSRF/CSP security, AST manipulation, Babel plugins, Node.js/Deno/Bun internals

**Videos & Books:**
- [JavaScript Engines — How V8 Works (JSConf EU)](https://www.youtube.com/watch?v=p-iiEDtpy6I)
- [Proxy & Reflect Meta-programming — Fun Fun Function](https://www.youtube.com/watch?v=AqgVLYpBWG8)
- [Eloquent JavaScript — free online book](https://eloquentjavascript.net)
- [You Don't Know JS — free on GitHub](https://github.com/getify/You-Dont-Know-JS)

**Projects:** Build a JS framework from scratch · Write a Babel plugin · WebAssembly integration with Rust

## 🚀 How to Use This

**Option 1 — View it live**
Just visit the live URL above. No installation needed.

**Option 2 — Run it locally**
```bash
# Clone the repo
git clone https://github.com/seleniumforchitta/js-tutorial-chitta.git

# Open in browser — no build step needed
open index.html
```

**Option 3 — Deploy your own copy**
Fork this repo → go to Settings → Pages → select `main` branch → save. Your own copy will be live at `https://your-username.github.io/js-tutorial-chitta/` in about 60 seconds.
